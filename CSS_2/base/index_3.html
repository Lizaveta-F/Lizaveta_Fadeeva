Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока: Занимает всю доступную ширину Должен присутствовать цвет фона Во всех "статьях" сделайте отсутп снизу от соседей равным .2em, при помощи свойства margin-botton

Статья 2

Размер блока: Занимает всю доступную ширину Должен присутствовать цвет фона Во всех "статьях" сделайте отсутп снизу от соседей равным .2em, при помощи свойства margin-botton. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.





Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
